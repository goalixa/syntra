# Planning Skill for Syntra

## Overview

This skill provides task planning domain expertise to enhance Syntra's PlannerAgent with decomposition patterns, estimation heuristics, and task prioritization.

## Capabilities

- **Task Decomposition**: Break down complex tasks into steps
- **Estimation**: Estimate effort and duration
- **Prioritization**: Rank tasks by impact/urgency
- **Dependency Analysis**: Identify task dependencies
