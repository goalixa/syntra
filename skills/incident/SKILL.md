# Incident Skill for Syntra

## Overview

This skill provides incident investigation domain expertise to enhance Syntra's IncidentAgent with evidence collection, diagnosis workflows, and remediation guidance.

## Capabilities

- **Evidence Collection**: Auto-collect logs, events, metrics, traces
- **Diagnosis**: Pattern matching against known incidents
- **Remediation**: Step-by-step fix suggestions
- **Timeline**: Build incident timeline from events
